# Changelog

All notable changes to the Sticky Notes extension will be documented in this file.

## [0.9] - 2026-03-14

### Added
- **Glassmorphism Improvements**: Added backdrop blur to the sticky note content area for a more premium, modern look.
- **Local Assets**: Bundled all icons as local SVGs to fix Content Security Policy (CSP) errors on sites like GitHub.
- **Changelog Page**: Added a dedicated changelog page to keep track of new features and fixes.

### Fixed
- **Header Visibility**: Fixed an issue where the sticky note header would disappear when collapsed (minimized).
- **CSP Compliance**: Removed dependencies on external Flaticon fonts and external Blob URLs.
- **UX Improvements**: Removed the hover "bounce" animation for a cleaner interaction feel.
