Release Notes

What's new in Sticky Notes

Version 0.11

Released on April 24, 2026

Added
  • Cloud Synchronization: Seamlessly sync your notes across devices using Google Sign-In.
  • Admin Dashboard: Built-in role-based access control with a dedicated dashboard for administrators.
Fixed
  • Search Stability: Fixed a bug where collapsing a domain group would reset active search results.
  • Authentication Redirects: Resolved Google OAuth blocking issues in the live Chrome Web Store environment.
Version 0.9

Released on March 14, 2026

Added
  • Glassmorphism Improvements: Enhanced the visual depth with backdrop blur on note content areas.
  • Local Assets: All UI icons are now bundled locally to ensure stability across all websites.
  • Dedicated Changelog: This page! Stay updated with the latest improvements.
Fixed
  • Collapsed Note Visibility: Fixed the bug where the note header would disappear when minimized.
  • CSP Compliance: Resolved font loading issues on strict sites like GitHub.
  • Smoother Interaction: Removed the hover bounce for a more stable and professional feel.
